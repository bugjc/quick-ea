## quick-ea
[快速开始](https://github.com/bugjc/quick-ea/wiki/%E5%BF%AB%E9%80%9F%E5%BC%80%E5%A7%8B)企业架构，关注业务代码的开发。  

## 端口约定
- API 网关：7900 ~ 8000
- 基础服务：8000 ~ 8100
- 监控应用：8200 ~ 8500
- 定制业务任务：8600 ~ 8800
- 数据分析任务：8900 ~ 9400
- 定制业务服务：9500 ~ 10000
